# sinricpro-website
This repository contains the Sinric Pro website http://sinric.pro code


### How to host
1. Clone this repo.
2. Install [NodeJS](https://nodejs.org/) and [NPM](https://www.npmjs.com/get-npm).
3. Run `npm i` to install dependencies.
4. Run `npm start`.
5. Goto http://localhost:3000/
