Sinric Pro कैसे काम करता है

1

अपना Sinric Pro खाता बनाएँ

अपने बहुत ही Sinric Pro खाते के लिए साइन अप करें। यह सरल और मुफ़्त है।

2

स्मार्ट होम डिवाइसेस जोड़ें

Githb रेपो से उदाहरण कोड डाउनलोड करें और वाईफाई, डिवाइस आईडी और क्रेडेंशियल बदलें

3

Control

Sinric Pro App, Alexa Skill, Google Action, IFTTT का उपयोग करें या एपीआई का उपयोग करके अपना खुद का निर्माण करें।

क्यों Sinric Pro?

प्रयोग करने में आसान

इसे सेटअप करने में 5 मिनट से कम समय लगता है

  चरण 1: एक खाता बनाएं और लॉगिन करें
  चरण 2: एक नया उपकरण जोड़ें
  चरण 3: Github से उदाहरण कोड डाउनलोड करें Github
  चरण 4: APP, ALEXA, GOOGLE HOME,IFTTT या API का उपयोग करके नियंत्रण
